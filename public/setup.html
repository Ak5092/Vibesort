VibeSort setup

~10 minutes · free · users just tap Connect after this

1
Deploy to Vercel

Go to vercel.com/new → sign up free with GitHub

Click Browse or drag the vibesort/ folder → click Deploy

✦ Vercel gives you a URL like https://vibesort-abc123.vercel.app — copy it

2
Register your Spotify app

Go to developer.spotify.com/dashboard → Create app

FieldValue
App nameVibeSort (anything)
DescriptionPlaylist sorter (anything)
Redirect URIhttps://YOUR-URL.vercel.app/api/auth?action=callback
APIs✓ Web API

After saving → Settings → copy Client ID and Client Secret

3
Add environment variables in Vercel

Vercel project → Settings → Environment Variables → add all three:

Variable nameValue
SPOTIFY_CLIENT_IDFrom Spotify dashboard
SPOTIFY_CLIENT_SECRETFrom Spotify dashboard
REDIRECT_URIhttps://YOUR-URL.vercel.app/api/auth?action=callback

After adding → Deployments → click the three dots → Redeploy

4
Verify it works

Open this URL in your browser — it should show all three as true:

https://YOUR-URL.vercel.app/api/auth?action=ping

If any show false → check spelling of env var names, then redeploy

5
Add to iPhone

Open your Vercel URL in Safari (not Chrome)

Tap the Share button → "Add to Home Screen" → Add

VibeSort appears as a full-screen app icon 🎉

🔧 Debugging tip
If Connect Spotify does nothing or errors, visit /api/auth?action=ping on your deployment.
All three values must be true. If not, re-check env vars and redeploy.
